# Throttle Talks

> **Note**: This project is currently under active development.

**Throttle Talks** is a premium, community-driven platform designed exclusively for automotive enthusiasts. It bridges the gap between social networking and practical utility, offering a dedicated space for builders, drivers, and fans to connect, showcase their machines, and organize local events.

---

## Overview

Built with a focus on aesthetic minimalism and distinctive typography, Throttle Talks provides a distraction-free environment where the content—cars, builds, and discussions—taking center stage. The application leverages a high-contrast monochrome design system to ensure a premium user experience across all devices.

## Core Features

### 🏎️ The Garage
A digital showroom for your pride and joy.
- **Detailed Build Logs**: Track modifications, service history, and specs.
- **Showcase Mode**: High-fidelity image sliders to present your vehicle's transformation.

### 📍 Meetups & Events
Discover and organize local car culture.
- **Local Discovery**: Find car meets, track days, and cruises near you.
- **Event Management**: RSVP handling and location mapping.

### 💬 Community Feed
A rich, mixed-media stream tailored to petrolheads.
- **Reviews**: In-depth user reviews of parts and vehicles.
- **Mod Showcases**: Before/after comparisons and installation guides.
- **Polls & Q&A**: Community voting and technical support.

### ⚡ Real-time Communication
- **Direct Messaging**: Private chats with other enthusiasts.
- **Status Indicators**: See when friends are online or at the track.

---

## Technology Stack

The project is engineered for performance and scalability using modern mobile technologies:

- **Framework**: [Expo](https://expo.dev/) (React Native)
- **Language**: TypeScript
- **State Management**: Zustand
- **Navigation**: React Navigation 7
- **UI/UX**: Custom Design System, Lucide Icons, Expo Blur
- **Animations**: React Native Reanimated
